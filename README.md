# Lucidia Metaverse

Complete 3D metaverse platform deployed to lucidia.earth

## Features
- Three.js 3D rendering engine
- Cannon.js physics simulation  
- Procedural cityscape generation
- Floating islands with portals
- Particle effects and dynamic sky
- WebXR/VR support
- Multiplayer chat (Socket.io ready)

## Deployment
Automatically deployed to Cloudflare Pages via GitHub Actions.
